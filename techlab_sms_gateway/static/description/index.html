TechLab SMS Gateway

Generic SMS Gateway integration with queue_job support for Odoo 18 CE

Features

Multiple SMS Gateways
Configure multiple SMS gateways with different APIs and choose which one to use.
Asynchronous Processing
Uses queue_job for reliable, asynchronous SMS sending with proper error handling.
SMS Templates
Create reusable SMS templates with dynamic content using placeholders.
Complete Logging
Track all sent messages with detailed logs including responses and errors.
CRM Integration
Send SMS directly from Partner and Lead records with automatic chatter logging.
Test Tools
Built-in test wizard to verify gateway configurations before going live.

Usage

  1. Configure your SMS gateways in the SMS → Gateway SMS menu
  2. Create SMS templates in SMS → Template SMS menu
  3. Send SMS from Partner or Lead records using the "Send SMS" button
  4. Monitor sent messages in SMS → Log SMS menu
